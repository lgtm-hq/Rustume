Rustume does not reserve functionality for a paid tier. You can deploy the open-source connected
application yourself, including synchronization, sharing, history, and API access. Rustume Cloud
charges for a ready-to-use hosted deployment: sign in, synchronize resumes, and let the service
operate the infrastructure.

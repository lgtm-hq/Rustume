## What hosted access pays for

A hosted subscription funds operation of the ready-to-use Rustume Cloud service: application
hosting, managed storage, synchronization availability, backups, observability, security updates,
and billing support. It does not provide feature entitlements beyond those available in
self-hosted deployments.

## Billing

[Paddle](https://developer.paddle.com/) acts as Merchant of Record for hosted-service checkout,
invoicing, and applicable tax handling. Billing state controls access to the Rustume-operated
deployment only; it is not a feature entitlement system for the application.
